# 数据结构与算法

## 数组与链表

| 对比项 | 数组 | 链表 |
|--------|------|------|
| 内存 | 连续 | 不连续 |
| 随机访问 | O(1) | O(n) |
| 插入删除 | O(n) | O(1) |

常见面试题：反转链表、合并有序链表、判断环。

## 栈与队列

- **栈**：后进先出（LIFO），用于括号匹配、表达式求值、DFS
- **队列**：先进先出（FIFO），用于 BFS、滑动窗口

## 二叉树

二叉树遍历方式：

- 前序：根 → 左 → 右
- 中序：左 → 根 → 右（BST 中序遍历得有序序列）
- 后序：左 → 右 → 根
- 层序：按层 BFS

常见考点：最大深度、对称二叉树、最近公共祖先。

## 排序算法

| 算法 | 平均时间 | 空间 | 稳定性 |
|------|----------|------|--------|
| 冒泡 | O(n²) | O(1) | 稳定 |
| 快排 | O(n log n) | O(log n) | 不稳定 |
| 归并 | O(n log n) | O(n) | 稳定 |
| 堆排 | O(n log n) | O(1) | 不稳定 |

快排是分治思想代表，注意最坏情况退化为 O(n²)。

## 动态规划

动态规划三要素：

1. **状态定义**：`dp[i]` 表示什么
2. **状态转移**：`dp[i]` 如何从之前状态推导
3. **初始条件**：边界值

经典题目：爬楼梯、最长递增子序列、背包问题。
